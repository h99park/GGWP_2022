# GGWP : Good Gamers Well Paired
게이머들간 인삿말로 사용되는 GGWP, 'Good Game Well Played'의 줄임말에서 앞글자를 따온 저희의 서비스는 Good Gamers Well Paired를 모토로 삼아 사람들이 함께 게임을 할 수 있도록 도움을 줄 웹페이지 서비스입니다.
